SETUP / HOW-TO

Patient Input

  1. Once the Mini C is on and booted up then input:
    • Last Name
    • First Name
    • MRN #
    • Doctor
    • Procedure
  2. Press the image tab on the right side of the screen to get read to take x-rays

Inputing the Procedure is optional but can be useful to have saved if you need to go back to find a case in the future

Saving / Printing Images

  1. After taking an X-ray, press the Save button that is on the Mini C-Arm above the keyboard
  2. To SELECT MULTIPLE IMAGES you can HOLD the Ctrl button on the keyboard and then begin to tap all of the images that you would like to print
  3. To print the images that you have selected just press the "Local Print" button and it will send the images to the printer below

Searching for Previous Case

  1. Press the "Change Exam" button that is at the top right of the image page
  2. You will see a list of all the patients that have been saved in the past over around a one year period. You can look through by date to find the patient that you need
  3. Select that patient to view their images and save or print whatever you need

Image Exporting

  1. To Export images to a USB you will plug a USB into the port that is on the Mini C-Arm above the keyboard
  2. Select all images you need to save holding Ctrl and tapping each image
  3. Prior to sending make sure to press "option" then press "BMP" for the file type on the images
  4. Send over to the USB by pressing the "Export"

Image Editing

  1. You can edit the contrast, brightness and sharpness of an image that has been taken with controls that are on the keyboard
  2. Select the image you want to edit by tapping it
  3. Play with the various settings on the keyboard until you get the image how you like it
  4. Make sure to press the "Save" button to add this edited image to the image list

Paper Changing

  1. To change the paper just press the "Open" button that is located on the right side of the printer
  2. The paper cartrige tray will open and you can pull the empty roll out and repalce with the new roll
  3. Pull a little but of the paper out so that it will feed properly
  4. I usually press the "Feed" button to test the paper feeding prior to printing

Common Problems

  1. People will forget to save an edited image so make sure to save if in doubt
