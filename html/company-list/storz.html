AIDA

Patient Input

  1. Press the patient tab at the bottom and input the information:
    • First Name, Last Name, MRN, Doctor
  2. Press the "Capture button at the bottom to switch to the screen to see images and videos that are captured.

Aida Printing

  1. Make sure that the printer is ON
  2. Press the "Finish" button on the bottom right
  3. Make sure that you have the Standard format box checked so that you print with the correct image format
  4. Press "Save" to send the images to the printer

Aida Re-Printing Old Case

  1. Press the File Cabinet icon in the top right side of the screen
  2. From here you will see all the cases from the past and to select one you just press the checkbox
  3. In the bottom right side of the screen you then have the option to either Open, Print, or Export the case so choose which you need
  4. If you press Print then you will be directed to a page where you will choose the format you are looking for. Choose the STANDARD format.

Getting to the Desktop of the Aida

  1. Press the little human icon and logout
  2. Input the username and password for the admin account.
    • Username: or1admin
    • Password: or1admin
  3. Press "OK" then press the Wrench Icon to get to the system setting page
  4. Press "System" then press "Windows Desktop"
  5. After logging out you login under the or1admin account and the password is the SAME
  6. From here you are in the desktop and you can then do whatever it is that you need to do
