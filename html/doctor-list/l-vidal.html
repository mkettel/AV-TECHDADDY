DR. LESLIE VIDAL

Below are videos and step-by-step directions on how to set up cases for Dr. Vidal and how to use / troubleshoot equipment that she commonly uses.

HIP SCOPE CASES

Smith & Nephew Tower Setup

(make a L.Vidal Hip video)

Dr. Vidal and her team use the Smith and Nephew tower for her Hip Scope cases because she likes the 560 box that is on the tower with her name on it. **Make sure to use the tower with her name on it if possible**

  1. Plug in the tower to a RED outlet if possible
  2. Turn on the tower and check that master power switch located on the side of the tower is ON
  3. Press the power button that is on the 660 box so you can input the patient information
  4. While everything is booting up, Turn on the printer on the bottom rack by pressing the power button
  5. Input the following information into the 660:
    • First / Last Name
    • MRN
    • Doctor Name
    • Procedure Type
  6. Press "Save" to save that patient and you can then select that patient to start the case
  7. Make sure to plug in the HDMI / DVI Cable into either the Boom or the Wall Aux
  8. Go over to the Nurse Touchpanel and make sure to route BOOM AUX or BOOM 4n1 (depending on which one you used) to the monitors so that you can put the scope video on all the monitors you like

OTHER SCOPE CASES

Arthrex Tablet Patient Input

Dr. Sterett and his team are tech saavy and they will use the Arthrex tablets and software for the images, videos and dictations of the case.

  1. Press the "user normal" button on the tablet to log into the software
  2. Press "New Patient" button in the upper right part of the screen
  3. Input the following information:
    • First / Last Name
    • MRN
    • Doctor Name
    • Procedure Type
  4. Press either "SAVE" or "START CASE" depending on if you want to input more patients or start that case directly
  5. Make sure to go to the nurse touchpanel and press "ARTHREX" and tap all the screens that you want the scope video routed to

Re-Printing a Previous Case

  1. Press the button that has the image of the file folder in upper right side
  2. From there you can select the patient you're looking for and press "review"
  3. The patient page will open and you can then print or save to a USB
  4. To print you just press the "Print" butto below and then you will be asked to chose the images per page and copy amount
