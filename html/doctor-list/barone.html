DR. BARONE

Below are videos and step-by-step directions on how to set up cases for Dr. Barone and how to use / troubleshoot equipment that she commonly uses.

Faxitron Setup / Login

  1. Plug in Faxitron to red wall outlet
  2. Plug in Ethernet cable into jack under nurse's station labeled "PACS"
  3. Turn on Faxitron holding power button down
  4. Select Tech Manager account and input Password that is listed above the screen
  5. Faxitron Program will auto-start and open to the patient input page.
  6. See next video for following steps...

Faxitron Patient Input

  1. Press "Patient List" button
  2. Press "New"
  3. Input Last Name, First Name, DOB, MRN, Procedure
  4. Press "Open"
  5. Press: Grey "Not Ready" button to which will shift to Green "Ready" message

Sending Images to PACS

  1. Press the "Archive / Export" button
  2. Select ALL of the images by tapping each one
  3. Press "ARCHIVE" to send the images over
  4. Problems:
    1. If error message appears in bottom left corner then make sure the ethernet cable has beet plugged into the port. Re-send images over to try again.
    2. If the issue still persists, then reach out to the Vail Health PACS admin Joe Webb and he will upload the images manually:
      • Phone: 970-471-5435
      • Email: webb@vailhealth.org

Setting up the SPY system

  1. Plug in the cart into a RED outlet
  2. Grab the HDMI cable and plug the cable into the plate underneath nurse station or into the BOOM
  3. Turn on the SPY system box by pressing the GREEN button
  4. Once on, go to the nurse touchpanel and make sure to route the SPY to the monitors that you need

Setup Checklist

  1. Is the Faxitron plugged in and the ethernet cable plugged into the PACs port in the wall?
  2. Is the Patient information plugged into the Faxitron?
  3. IF Spy is needed, is the cart plugged in and powered on?
  4. Is the HDMI cable on from the SPY plugged into the Wall Aux or Boom Aux?
